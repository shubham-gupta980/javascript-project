## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/shubham-gupta980/javascript-project.git
```

2. Navigate to the project folder

```bash
cd javascript-project
```

3. Open any project folder.

4. Open the `index.html` file in your browser or use the VS Code Live Server extension.
